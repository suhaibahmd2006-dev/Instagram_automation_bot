# Insta-AI Automation Bot

A versatile Python-based Instagram automation agent designed for productivity, media management, and educational cybersecurity awareness.

## 🚀 Features
- **AI-Powered DMs:** Intelligent conversation handling via OpenRouter.
- **Media Generation:** Create custom AI-generated images on demand.
- **Voice Synthesis:** High-energy text-to-speech voice notes powered by Fish Audio.
- **Music & Media:** Download and listen to songs or full tracks easily.
- **Utility Tools:** Dictionary lookups and helpful system commands.
- **Cybersecurity Awareness:** Educational info on mobile security, network protection, and defense.

## 🤖 Available Commands
| Command | Description |
| :--- | :--- |
| `-menu` | Shows the full list of available bot commands. |
| `-ai <question>` | Ask the AI any question. |
| `-generate <desc>` | Create an AI-generated image from your description. |
| `-tts <text>` | Convert text to a custom voice note. |
| `-music <name>` | Download and process a song track. |
| `-vn <name>` | Send a specific audio track as a voice note. |
| `-hack <topic>` | Educational info on security, DDoS, and mobile protection. |
| `-urban <word>` | Fetch a definition from the Urban Dictionary. |

## 🛠 Prerequisites
- **Python 3.10+**
- **FFmpeg:** Essential for audio/media processing.
  - *Install via Termux:* `pkg install ffmpeg`
- **API Keys:** You will need valid keys for OpenRouter and Fish Audio.
